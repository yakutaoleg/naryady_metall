# Telegram Bot — UX и логика работы

## Концепция

Бот для рабочих и мастеров цеха металла. Всё управление — только через кнопки, без команд.
Рабочий обычно имеет одну специализацию и видит только свои задачи.
Мастер/Администратор — видит весь цех.

---

## Экраны рабочего

### 1. Главное меню (`/start`)

```
👷 ГАНЖА Иван
🔧 Специализация: ПЛАЗМА

Выберите действие:

[ 📋 Задачи в работу ]  [ ✅ Выполнено сегодня ]
```

### 2. Список задач

```
📋 Задачи | ПЛАЗМА

❗ Обязательные:
  • ПС1 — СТ1 × 8 шт
  • ПС5 — СТ1-А × 2 шт

Остальные 🔒 (после обязательных):
  • ПС2 — СТ1 × 4 шт
  • ПС7 — Р1 × 1 шт

👇 Нажмите на задачу:
[ ❗ ПС1 — СТ1 × 8     ]
[ ❗ ПС5 — СТ1-А × 2   ]
[ 🔒 ПС2 — СТ1 × 4     ]
[ 🔒 ПС7 — Р1 × 1      ]
[ ← Главное меню        ]
```

**Логика блокировки:**
- Пока есть невыполненные обязательные — необязательные заблокированы (🔒)
- При нажатии на 🔒 задачу: pop-up "Сначала выполните обязательные: ПС1, ПС5"
- Как только все обязательные выполнены — 🔒 снимается автоматически

### 3. Карточка задачи

```
❗ [ПЛАЗМА] ПС1
Элемент: СТ1
Проект: Орша теплосети
Кол-во: 8 шт
Масса ед.: 1.47 кг  |  Масса всего: 11.76 кг
Дата план: 20.06.2026
Обязательная: ДА

💰 Тариф: 1.25 руб/кг
💵 К оплате: 5.99 руб

[ ✅ ВЫПОЛНЕНО ]  [ 🚫 БЛОК ]
[ 📄 Чертёж               ]   ← только если прикреплён
[ ← К задачам             ]
```

### 4. После ВЫПОЛНЕНО

Статус задачи → ВЫПОЛНЕНО, дата факт → сегодня.
Бот сразу возвращает к обновлённому списку задач.

**Защита от двойного нажатия:** повторный клик ВЫПОЛНЕНО показывает alert "Задача уже отмечена как выполненная."

### 5. БЛОК

Бот просит написать причину текстом → сохраняет в поле `comment`.
После → кнопка "← К задачам".

**Доступ:** рабочий может поставить БЛОК только на свою задачу (проверка `executor`).

### 6. Чертёж

Рабочий нажимает "📄 Чертёж" → бот скачивает PDF из Google Drive и отправляет в чат.

**Доступ:** чертёж доступен только исполнителю задачи (проверка `executor`).

### 7. Выполнено сегодня

```
✅ Выполнено сегодня (2):

✅ ПС1 — СТ1 × 8 шт  |  5.99 руб
✅ ПС5 — СТ1-А × 2 шт  |  3.20 руб

💵 Итого за день: 9.19 руб

[ ← Главное меню ]
```

## Экраны мастера / администратора

### 1. Главное меню мастера

```
👷 ИВАНОВ Пётр  |  Мастер
Проект: Орша теплосети

[ 🚫 Блоки ]         [ 📊 Выработка ]
[ 📁 Проекты ]       [ 🔄 Синхронизировать ]

Последняя синхронизация: 14:32
```

### 2. Экран "🚫 Блоки"

Все активные блокировки по всем проектам с кнопкой "Снять блок".

**Защита от двойного снятия:** повторный клик показывает alert "Блок уже снят другим мастером."

### 3. Экран "📊 Выработка"

```
📊 Выработка — Неделя

Иванов А.      — 1 240.50 руб
Петров В.      —   980.00 руб
Сидоров К.     —   760.20 руб

[ Сегодня ]  [ Неделя ]  [ Месяц ]
[ ← Назад ]
```

Фильтр по периоду: Сегодня / Неделя / Месяц.

### 4. Управление проектами

Добавить проект по ссылке Google Drive, архивировать, восстановить.

## Идентификация

- Поиск по `telegram_username` в таблице `employees`
- Специализация берётся из `employees.specialization`
- Роль (worker / master / admin) из `employees.role`
- Если пользователь не найден — бот просит обратиться к руководителю

## Тарифы

Хранятся в таблице `tariffs` (синхронизируются из вкладки ТАРИФЫ в Google Sheets).

| Вид работы | Диапазон массы ед. (кг) | Ставка      |
|------------|--------------------------|-------------|
| ПЛАЗМА     | 0 — 1                    | 1.25 руб/кг |
| ПЛАЗМА     | 1 — 5                    | 0.51 руб/кг |
| ПЛАЗМА     | 5 — 10                   | 0.222 руб/кг|
| ПЛАЗМА     | 10+                      | 0.10 руб/кг |
| ПИЛА       | любая                    | 0.15 руб/кг |
| СБОРКА     | любая                    | 0.25 руб/кг |
| СВАРКА     | любая                    | 0.234 руб/кг|
| ПОКРАСКА   | любая                    | 0.12 руб/кг |

Сумма к оплате (`payment_sum`) берётся из Google Sheets как есть.
Тариф в карточке задачи — справочная информация.

## Защита статусов

Sync из Google Sheets **не перезаписывает** статусы ВЫПОЛНЕНО и БЛОК.
Только сам рабочий через бот может их установить.
